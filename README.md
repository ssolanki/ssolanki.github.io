## Quick Start

Install Node.js and then:

```sh
$ cd rentomojo
$ sudo npm -g install grunt-cli
$ npm install
$ bower install
$ grunt watch
```

Finally, open `file:///path/to/rentomojo/build/index.html` in your browser.

Happy hacking!
